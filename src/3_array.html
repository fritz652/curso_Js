ARRAY - ARREGLO

Un Array es una lista de datos que se expresna en corchetes y entre comillas

let hermanos = ["YY","PP","FF","JJ"]

let dias = ["lunes","martes","miercoles","jueves","viernes","sabados","domingos"]

les vocales =["a","b","c","d","e"]


Un Array es uno de los tipos de datos mas importantes

Recuerda: cada elemento de un Array se puede llamar con [i], por ejemplo vocales[0] es el equivalente a "a" y vocales [4] es el equivalente a "u"
