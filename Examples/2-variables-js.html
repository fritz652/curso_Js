"VARIABLES"

Una variable es un contenedor de valor.

antes se usaba la palabra "var" para declarar una variable, actualmente se utiliza "let"

Ejemplo: let nombre = "carlos"

donde: let declaclara la variable nombre, quien contiene a "carlos"
