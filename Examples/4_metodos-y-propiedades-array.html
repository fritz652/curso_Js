METODOS Y PROPIEDADES ARRAY

Propiedades de los Array

  • ___.length()

    • Te ayuda a saber cuantos elementos tiene un Array

    Ejemplo:

    let amigos = ["Carlos","Pedro","Alex"];

    documet.write(amigos.length;

    te arrojara como resultado el nro 3

  • ___.push()

    • Te ayuda a agregar un elemento al array

    Ejemplo:

    let amigos = ["Carlos","Pedro","Alex"];

    amigos.push("Luis","Arturo");

    te arrojara como resultado

    amigos = ["Carlos","Pedro","Alex","Luis","Arturo"]

  • ___.pop()

    • Te ayuda a eliminar el ultimo elemento del array

    Ejemplo:

    let amigos = ["Carlos","Pedro","Alex"];

    amigos.pop();

    te arrojara como resultado

    amigos = ["Carlos","Pedro"]

  • ___.concat()

    • Te ayuda a concatenar dos o mas array

    Ejemplo:

    let amigos = ["Carlos","Pedro","Alex"];

    let amigos2 = ["Juan","Marlon"];

    amigos3 = amigos.concat(amigos2);

    document.write(amigos3);

    te arrojara como resultado

    ["Carlos","Pedro","Alex","Carlos","Pedro"]

    Siempre que le antepondas un document.write(), para que sea visible en la pagina

  • ___.join()

    • Te ayuda a modificar las comas de un array

    Ejemplo:

    let amigos = ["Carlos","Pedro","Alex"];

    document.write(amigos.join(":"));

    ["Carlos":"Pedro":"Alex"]

  • ___.sort()

    • Te ayuda a ordenar en forma alfabetica el array

    Ejemplo:

    let amigos = ["Carlos","Pedro","Alex"];

    let ordenados =amigos.sort();

    document.write(ordenados)

    te arrojara como resultado

    Alex,Carlos,Pedro

  • ___.reverse()

    • Te ayuda a ordenar en forma de reversa al array

    Ejemplo:

    let amigos = ["Carlos","Pedro","Alex"];

    let ordenados =amigos.sort();

    document.write(ordenados)

    te arrojara como resultado

    Alex,Pedro,Carlos
